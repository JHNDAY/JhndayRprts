Reference Notes
⚠ Run Installer as Administrator
Both VB-Cable installers must be run as Administrator or the driver will not install. Right-click the .exe → Run as Administrator. A restart is required after each install before the devices appear in Sound Settings.
⚠ No Local Monitoring
With Zoom's speaker set to CABLE Input, no audio plays through the PC's speakers. This machine is a pure pass-through. Monitor the meeting from the host machine or another device.
ℹ Disable Zoom Audio Processing
In Zoom → Settings → Audio, disable Suppress background noise and Suppress persistent noise. Zoom's processing will otherwise filter or suppress Claude's TTS voice before it reaches participants.
ℹ VB-Cable Naming Logic
VB-Cable naming is counterintuitive: CABLE Input is where apps send audio into the cable; CABLE Output is where apps receive audio from the cable. Think of Input/Output from the cable's perspective, not the app's.
