Understanding AI in Conversation

A 4-Part Interactive Learning Series for BOLLI

Session Structure Template (90 minutes each)

10 min - Welcome, Agenda & Interactive Poll
30 min - Core Content with Visual Aids
20 min - Live Demonstrations
20 min - Guided Practice & Activities
10 min - Q&A & Open Discussion
1

Generative AI vs. RAG Technology

Learning Objectives:

  • Understand how generative AI creates content
  • Grasp the concept and importance of RAG
  • Identify use cases where RAG excels

Key Topics:

Pattern Recognition Hallucinations Real-time Accuracy Source Verification

Interactive Elements

  1. Live Demo: Compare pure AI vs. RAG responses
  2. Group Exercise: Identify RAG use cases
  3. Poll: "Which would you trust for medical info?"
Examples to show: ChatGPT, Claude, Perplexity, Enterprise chatbots
2

AI Memory and Conversations

Learning Objectives:

  • Understand how AI stores conversation context
  • Learn about session vs. long-term memory
  • Recognize privacy and practical implications

Key Topics:

Session Memory Custom Instructions Privacy Personalization

Interactive Elements

  1. Live Experiment: Show memory persistence
  2. Hands-on: Set up custom instructions
  3. Discussion: What should AI remember/forget?
Demonstrations: ChatGPT memory features, clearing memory, context comparison
3

Understanding and Talking with AI Avatars

A live demonstration of custom AI personas joining your Zoom meeting

Key Components:

  • Historical AI Avatars: Alan Turing, Ed Murrow, Claude Shannon
  • Separate Zoom logins for each avatar persona
  • ChatGPT Projects with defined personalities and rules
  • Audio-only sessions for real-time voice interaction

Technologies Used:

ChatGPT Projects Voice AI Zoom Integration Character Design

Interactive Elements

  1. Avatar Introduction: Meet Turing, Murrow, and Shannon
  2. Real Conversation: Watch them discuss AI's future
  3. Audience Q&A: Participants ask historical figures questions
Setup includes: Custom ChatGPT projects, Zoom credentials, voice configuration, personality prompts
4

The Power of Visualizing ChatBot Responses

"A picture is worth a thousand words - especially if you can talk with it"

Learning Objectives:

  • Understand Claude's HTML artifact capabilities
  • Learn to transform text responses into visual formats
  • Master iterative refinement through conversation

Key Features:

Interactive Dashboards Visual Summaries Dynamic Charts Real-time Editing

Interactive Elements

  1. Live Creation: Generate visual from complex prompt
  2. Iterative Refinement: Modify through conversation
  3. Component Analysis: Understand each element
Example visualizations: Process workflows, data comparisons, learning materials, interactive reports

Technical Setup Recommendations

For Zoom:

  • Enable auto-transcription
  • Use polls and breakout rooms
  • Share screen with "optimize for video clip"
  • Record to cloud for easy sharing

For Artifact Creation:

  • Use Otter.ai or Zoom's transcript for initial text
  • Edit with Descript for video highlights
  • Create guides in Canva for visual appeal
  • Host materials on a simple website or LMS

Engagement Strategy Timeline

Before Each Session

  • Send pre-session survey about experience level
  • Share a primer article or video
  • Create anticipation with a teaser question

During Sessions

  • Start with an AI-related icebreaker
  • Use analogies relevant to your audience
  • Encourage active chat participation
  • Take "comfort breaks" for processing

After Sessions

  • Send follow-up email with materials
  • Create a discussion forum
  • Offer "office hours" for questions
  • Share participant success stories
