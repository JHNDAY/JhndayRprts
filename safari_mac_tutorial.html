Finding Your Keys

EASY
Learn to identify and locate the special Mac keyboard keys (⌘, ⌥, ⇧, ⌃)

Understanding Mac Key Symbols

Look at your keyboard and find these four keys:
Symbol Key Name Where to Find It
⌘ Command Bottom row, immediately left and right of the spacebar. Has a ⌘ symbol and/or says "command"
⌥ Option Bottom row, next to the Command keys. Says "option" or "alt"
⇧ Shift Left and right sides, above Command. Has an upward arrow ⇧ and says "shift"
⌃ Control Bottom left corner, says "control" or has "^" symbol
Practice Exercise 1: Key Location Drill

Let's practice finding these keys quickly. Do each of these WITHOUT pressing:

  1. Place your left index finger on the Command key (⌘)
  2. Now move it to the Option key (⌥)
  3. Find the Shift key (⇧) with your left pinky finger
  4. Locate the Control key (⌃) with your left pinky
  5. Repeat this 5 times until you can find each key without looking
Checkpoint: Can you do this?
  • I can find the Command key (⌘) without looking at my keyboard
  • I can find the Option key (⌥) quickly
  • I know where both Shift keys (⇧) are located
  • I can locate the Control key (⌃)
💡 Memory Tip: Most shortcuts use Command (⌘) as the main key. Think of it as the "Mac action key" - it's your new best friend!

Opening & Managing Safari Tabs

EASY
Master opening, closing, and switching between tabs efficiently for managing multiple websites during classes

Why This Matters

During Zoom meetings, you'll often need multiple websites open - class materials, research sources, or reference documents. Tabs keep everything organized in one window!

1 Opening a New Tab

Press: ⌘+T

(Hold down Command, then tap the T key, then release both)

Practice Exercise 2A: Opening Multiple Tabs
  1. Make sure Safari is open (click the compass icon in your dock)
  2. Press ⌘ + T to open a new tab
  3. Notice the new empty tab that appears at the top
  4. Practice this 5 times until you have 6 tabs open
  5. Look at the tab bar - see how many tabs you created!
2 Closing a Tab

Press: ⌘+W

Practice Exercise 2B: Closing Tabs
  1. Click on any tab to make it active
  2. Press ⌘ + W to close it
  3. Notice how the tab disappears
  4. Close all but one of your tabs using this shortcut
🎯 Real-World Example: Imagine you're in a BOLLI discussion class. You have tabs open for: the Zoom meeting, class readings, a research article, and your notes document. Using ⌘ + T and ⌘ + W lets you quickly open new sources or close tabs you're done with!
3 Switching Between Tabs

Press: ⌘+1 for first tab, ⌘+2 for second tab, etc.

Practice Exercise 2C: Tab Navigation
  1. Open 4 new tabs (⌘ + T four times)
  2. In each tab, type a different website:
    • Tab 1: www.brandeis.edu
    • Tab 2: www.nytimes.com
    • Tab 3: www.theguardian.com/us
    • Tab 4: www.google.com
  3. Practice jumping between tabs:
    • Press ⌘ + 1 - You should see Brandeis
    • Press ⌘ + 3 - You should see The Guardian
    • Press ⌘ + 2 - You should see NY Times
  4. Practice this until you can quickly jump to any tab!
Checkpoint: Tab Management Skills
  • I can open a new tab without thinking about it
  • I can close the current tab quickly
  • I can jump to any tab using the number keys
  • I feel comfortable managing multiple tabs
⚠️ Common Mistake: Make sure you're pressing the Command key BEFORE you press the letter or number. Hold Command, tap the other key, then release everything. Don't try to press them at exactly the same time!

Creating & Managing Bookmarks

EASY
Save important websites and organize them in folders for quick access to course materials

Why Bookmarks Matter for BOLLI

You'll want quick access to your Zoom links, course materials, BOLLI homepage, and frequently referenced websites. Bookmarks save you from typing URLs repeatedly!

1 Saving Your First Bookmark

Press: ⌘+D

Practice Exercise 3A: Bookmark a Website
  1. Open Safari and go to www.brandeis.edu
  2. Once the page loads, press ⌘ + D
  3. A dialog box will appear - this is the bookmark window!
  4. Notice it shows:
    • The page title (you can edit this to something shorter)
    • Where to save it (we'll organize this next)
  5. Click "Add" to save your bookmark
💡 Naming Tip: Give your bookmarks short, clear names. Instead of "Brandeis University Official Website Homepage", try "BOLLI Home" or "Brandeis". You'll thank yourself later!
2 Opening Your Bookmarks Sidebar

Press: ⌘+⌥+B

(Hold Command and Option together, tap B, release all)

Practice Exercise 3B: View Your Bookmarks
  1. Press ⌘ + ⌥ + B
  2. A sidebar appears on the left side of Safari
  3. Find the Brandeis bookmark you just created
  4. Click on it - it should take you to that page
  5. Press ⌘ + ⌥ + B again to hide the sidebar
  6. Practice opening and closing the sidebar 3 times
3 Creating a Bookmarks Folder
Practice Exercise 3C: Create Your BOLLI Folder
  1. Open the Bookmarks sidebar (⌘ + ⌥ + B)
  2. Right-click (or Control + click) anywhere in the sidebar
  3. Select "New Folder" from the menu
  4. Type "BOLLI Courses" as the folder name
  5. Press Return (Enter) to create the folder
  6. See your new folder appear in the list!
4 Saving Bookmarks to Your Folder
Practice Exercise 3D: Organize Your Bookmarks
  1. Go to these three websites and bookmark each one:
    • www.zoom.us
    • www.wikipedia.org
    • www.theguardian.com/us
  2. For each website:
    • Press ⌘ + D to bookmark
    • In the dialog, click the dropdown menu
    • Select "BOLLI Courses" folder
    • Click "Add"
  3. Open your sidebar and look inside your "BOLLI Courses" folder
  4. You should see all three bookmarks there!
🎯 Real-World Setup: Create these folders for optimal organization:
  • BOLLI Courses - Main folder with subfolders for each course
  • Current Events - News sources for discussion classes
  • Research Tools - Libraries, databases, reference sites
  • Zoom Links - All your recurring meeting links
5 Renaming Bookmarks
Practice Exercise 3E: Rename a Bookmark
  1. Open the Bookmarks sidebar
  2. Find one of your bookmarks
  3. Click ONCE on it to select it
  4. Click AGAIN (or press Return) - the name becomes editable
  5. Type a new, shorter name
  6. Press Return to save the new name
6 Deleting Bookmarks
Practice Exercise 3F: Delete a Bookmark
  1. Open the Bookmarks sidebar
  2. Click on a bookmark to select it
  3. Press the Delete key on your keyboard
  4. The bookmark disappears!
  5. Try this with the Weather.com bookmark (we can recreate it later)
Checkpoint: Bookmark Mastery
  • I can bookmark a page in 2 seconds using ⌘+D
  • I can open and close the bookmarks sidebar easily
  • I created a "BOLLI Courses" folder
  • I can save new bookmarks directly to a folder
  • I can rename bookmarks to shorter, clearer names
  • I can delete bookmarks I no longer need

Taking Screenshots

EASY
Capture screen images to save class materials, document instructions, or share what you see

Three Ways to Capture Your Screen

1 Capture Entire Screen

Press: ⇧+⌘+3

(Hold Shift and Command, tap 3, release all)

Practice Exercise 4A: Full Screen Capture
  1. Arrange your screen so you can see multiple windows
  2. Press ⇧ + ⌘ + 3
  3. You'll hear a camera shutter sound - that means it worked!
  4. Look at your Desktop - a new file appeared with today's date and time
  5. Double-click the file to open and view your screenshot
  6. Practice this 2 more times to get comfortable
2 Capture Selected Area

Press: ⇧+⌘+4

Then drag to select the area you want to capture

Practice Exercise 4B: Selective Capture
  1. Open Safari with this tutorial page visible
  2. Press ⇧ + ⌘ + 4
  3. Your cursor turns into a crosshair (+)
  4. Click and drag to draw a box around just the lesson title
  5. Release the mouse - screenshot taken!
  6. Find it on your Desktop and view it
  7. Practice capturing different parts of your screen:
    • Just one paragraph of text
    • A specific image or diagram
    • The Safari toolbar
💡 Pro Tip: If you press ⇧ + ⌘ + 4 and then change your mind, press Esc to cancel. The crosshair disappears and no screenshot is taken.
3 Capture a Specific Window

Press: ⇧+⌘+4, then press Spacebar

Practice Exercise 4C: Window Capture
  1. Open Safari and another application (like Mail or Notes)
  2. Press ⇧ + ⌘ + 4
  3. The crosshair appears
  4. Press the Spacebar
  5. Your cursor changes to a camera icon 📷
  6. Move the camera over different windows - they highlight in blue
  7. Click on the Safari window
  8. Only that window is captured - perfect!
  9. Try capturing different windows on your screen
🎯 BOLLI Use Cases:
  • Save class materials: Capture diagrams, charts, or text from presentations
  • Remember Zoom settings: Screenshot your working audio/video setup
  • Share tech issues: Capture error messages to send to tech support
  • Create study notes: Capture key slides or web content for later review
  • Document processes: Take step-by-step screenshots to remember procedures
4 The Screenshot Toolbar (Advanced)

Press: ⇧+⌘+5

Practice Exercise 4D: Explore the Toolbar
  1. Press ⇧ + ⌘ + 5
  2. A toolbar appears at the bottom of your screen
  3. Explore the options:
    • First three buttons: Different screenshot modes
    • Last two buttons: Screen recording options
    • "Options" button: Change where screenshots save
  4. Click "Options" and look at "Save to" - you can change from Desktop to Documents
  5. Click away from the toolbar to close it
Checkpoint: Screenshot Skills
  • I can capture my entire screen quickly
  • I can select and capture just part of my screen
  • I can capture a specific window cleanly
  • I know where to find my screenshots (Desktop)
  • I can open the screenshot toolbar for more options

Using Split Screen for Zoom Meetings

MEDIUM
View Zoom and Safari side-by-side to follow course materials while participating in discussions

Why Split Screen is Essential for BOLLI

Imagine being in a current events discussion while simultaneously viewing the NY Times article everyone is discussing, or following along with class slides while seeing your classmates in Zoom. Split screen makes this possible!

1 Understanding the Green Button
Practice Exercise 5A: Find the Green Button
  1. Open Safari (or any window)
  2. Look at the top-left corner of the window
  3. You'll see three small circles: red, yellow, and green
  4. Hover your mouse over the green button (don't click yet!)
  5. Notice how when you hover, a small menu appears
  6. Familiarize yourself with this button - it's your split screen control!
2 Creating Your First Split Screen
Practice Exercise 5B: Set Up Split Screen
  1. Preparation:
    • Open Safari
    • Open another application (Notes, Mail, or another Safari window)
    • Make sure both windows are visible on screen
  2. Start Split Screen:
    • Click and hold the green button on one window
    • Keep holding for about 1 second
    • You'll see options appear
  3. Choose Position:
    • Select "Tile Window to Left of Screen"
    • The window moves to the left half
    • Your other windows appear on the right
  4. Select Second Window:
    • Click on the window you want on the right side
    • It fills the right half automatically
    • You're now in split screen mode!
💡 Quick Method: Hold the ⌥ Option key and click the green button - this immediately tiles the window to fill half the screen!
Practice Exercise 5C: Perfect for Zoom Classes
  1. Open Zoom (or use zoom.us in Safari for practice)
  2. Open Safari in a separate window
  3. Put Zoom on the left:
    • Click and hold Zoom's green button
    • Select "Tile Window to Left of Screen"
  4. Click Safari to put it on the right
  5. Now you can:
    • See your class participants in Zoom
    • View class materials in Safari
    • Follow along without switching windows!
3 Adjusting the Split
Practice Exercise 5D: Customize Your Layout
  1. While in split screen, look at the center line between windows
  2. Move your cursor over this line - it becomes a resize cursor ↔️
  3. Click and drag the line left or right
  4. Try these layouts:
    • Equal split: 50/50 - good for two equally important windows
    • Zoom priority: 60/40 with Zoom larger - better for seeing faces
    • Safari priority: 40/60 with Safari larger - better for reading materials
  5. Find what works best for your eyes and workflow
4 Exiting Split Screen
Practice Exercise 5E: Exit Split Screen

Three ways to exit:

  1. Method 1 - Press Esc:
    • Simply press the Esc key
    • Both windows return to normal
  2. Method 2 - Green Button:
    • Click the green button on either window
    • Both windows return to normal size
  3. Method 3 - Full Screen:
    • Press ⌃ + ⌘ + F
    • Makes the current window full screen
    • The other window moves to a separate space
🎯 Recommended BOLLI Setup:

For Discussion Classes:

  • Left: Zoom (to see and hear classmates)
  • Right: Safari with course readings or research

For Lecture Classes:

  • Left: Zoom (to see presenter and slides)
  • Right: Notes app or Google Doc for taking notes

For Current Events:

  • Left: Zoom (for discussion)
  • Right: Safari with multiple tabs of news sources
Checkpoint: Split Screen Mastery
  • I can find and use the green button confidently
  • I can create a split screen in under 10 seconds
  • I can adjust the split to my preferred layout
  • I know multiple ways to exit split screen
  • I understand how to set up Zoom + Safari for class
⚠️ Troubleshooting Tips:
  • Green button does nothing: Try clicking and holding longer (1-2 seconds)
  • Windows won't tile: Make sure you're not in full screen mode first (press Esc)
  • Can't see both windows: Your screen might be too small - try using full screen with swipe between spaces
  • Split looks weird: Adjust the divider by dragging the center line

Essential Navigation Shortcuts

EASY
Master keyboard shortcuts for faster browsing: address bar, back/forward, refresh, find text, zoom
1 Jump to Address Bar

Press: ⌘+L

Practice Exercise 6A: Quick URL Access
  1. While in Safari, press ⌘ + L
  2. Notice the address bar at the top is now selected (highlighted in blue)
  3. Type: nytimes.com and press Return
  4. Press ⌘ + L again
  5. Type: theguardian.com/us and press Return
  6. Practice this until it feels natural
💡 Why This Matters: During Zoom classes, instructors often share URLs verbally. With ⌘ + L, you can instantly jump to the address bar and type the URL without reaching for your mouse!
2 Go Back and Forward

Back: ⌘+[
Forward: ⌘+]

Practice Exercise 6B: Navigate Like a Pro
  1. Visit 4-5 different websites (use ⌘ + L to jump to address bar for each)
  2. Press ⌘ + [ several times to go back through your history
  3. Press ⌘ + ] to go forward
  4. Practice navigating back and forth until you're comfortable
3 Refresh a Page

Press: ⌘+R

Practice Exercise 6C: Reload Content
  1. Go to a news website (like nytimes.com or bbc.com)
  2. Press ⌘ + R to refresh the page
  3. Notice the page reloads with any new content
  4. This is useful when:
    • A page isn't loading correctly
    • You want to see if content has updated
    • Something looks broken
4 Find Text on Page

Press: ⌘+F

Practice Exercise 6D: Search Within a Page
  1. Go to any long article or webpage (like Wikipedia)
  2. Press ⌘ + F
  3. A search box appears at the top of the window
  4. Type a word you want to find on the page
  5. Safari highlights all instances of that word
  6. Use the up/down arrows in the search box to jump between matches
  7. Press Esc to close the search box
🎯 BOLLI Research Tip: When reading a long academic article or policy document for class, use ⌘ + F to quickly find specific terms, names, or concepts your instructor mentioned. This saves huge amounts of scrolling time!
5 Zoom In and Out

Zoom In: ⌘++
Zoom Out: ⌘+-
Reset Zoom: ⌘+0

Practice Exercise 6E: Adjust Text Size
  1. Go to any webpage with text
  2. Press ⌘ + + three times
  3. Notice text gets larger - easier to read!
  4. Press ⌘ + - five times
  5. Text becomes smaller
  6. Press ⌘ + 0 to return to normal size
  7. Find your comfortable reading size and remember that zoom level
💡 Accessibility Tip: If you find text too small during Zoom meetings, remember you can zoom in on any shared screen or document using these shortcuts. Each website remembers your zoom preference!
Checkpoint: Navigation Skills
  • I can jump to the address bar instantly
  • I can navigate back and forward through my browsing history
  • I can refresh pages when needed
  • I can find specific text on long pages
  • I can adjust text size for comfortable reading

Copy, Paste, and Text Management

EASY
Master moving text and content around - critical for note-taking, research, and communication
1 Selecting Text
Practice Exercise 7A: Text Selection Techniques
  1. Open a new tab and go to any article (try wikipedia.org)
  2. Select a word:
    • Double-click on any word
    • The word becomes highlighted
  3. Select a paragraph:
    • Triple-click anywhere in a paragraph
    • The entire paragraph highlights
  4. Select custom text:
    • Click and drag across the text you want
    • Release when you have what you need
  5. Select all:
    • Press ⌘ + A
    • Everything on the page is selected
2 Copy and Paste

Copy: ⌘+C
Paste: ⌘+V

Practice Exercise 7B: Copy-Paste Workflow
  1. Open two applications:
    • Safari with any article
    • Notes app (use Spotlight: ⌘ + Spacebar, type "notes")
  2. In Safari:
    • Select an interesting paragraph
    • Press ⌘ + C to copy
  3. Switch to Notes (click on it)
  4. Press ⌘ + V to paste
  5. The paragraph appears in Notes!
  6. Repeat this 3 times with different paragraphs
🎯 BOLLI Note-Taking Workflow:

During a Zoom class discussion:

  1. Instructor mentions important point from article
  2. You find that section in Safari
  3. Select and copy the relevant text (⌘ + C)
  4. Switch to your Notes or Google Doc
  5. Paste it (⌘ + V)
  6. Add your own thoughts below

All without losing your place in the Zoom discussion!

3 Cut (Move) Text

Cut: ⌘+X

Practice Exercise 7C: Cut and Paste
  1. In your Notes app, type: "This is practice text for cutting and pasting."
  2. Select the word "cutting"
  3. Press ⌘ + X (it disappears!)
  4. Click at the end of the sentence
  5. Press ⌘ + V
  6. The word appears in its new location
  7. Practice moving words around in your sentence
💡 Cut vs Copy:
  • Copy (⌘ + C): Duplicates text - original stays, you paste a copy
  • Cut (⌘ + X): Moves text - original disappears, you paste it elsewhere
4 Undo and Redo

Undo: ⌘+Z
Redo: ⌘+⇧+Z

Practice Exercise 7D: Undo Practice
  1. In Notes, type: "I am learning keyboard shortcuts!"
  2. Press ⌘ + Z
  3. Your typing disappears! (Don't panic - this is undo)
  4. Press ⌘ + ⇧ + Z
  5. Your text comes back! (This is redo)
  6. Practice:
    • Type something, undo it
    • Delete something, undo it
    • Paste something wrong, undo it
⚠️ Undo is Your Safety Net: Made a mistake? Deleted something important? Accidentally pasted the wrong thing? Just press ⌘ + Z! You can undo multiple steps - just keep pressing it. Most apps remember your last 50+ actions.
5 Real-World Practice: Research Collection
Practice Exercise 7E: Complete Research Workflow

Scenario: You're researching a topic for tomorrow's BOLLI discussion

  1. Open Notes app
  2. Create a new note titled "Current Events Research"
  3. Open Safari with 3 tabs:
    • New York Times
    • Wikipedia on your topic
    • Another news source
  4. From each source:
    • Find key information
    • Select and copy it (⌘ + C)
    • Switch to Notes
    • Paste it (⌘ + V)
    • Type your reaction or question below
  5. Organize your notes by cutting and pasting sections
  6. If you make mistakes, undo them
  7. Practice this workflow until it feels smooth
Checkpoint: Text Management Mastery
  • I can select text using multiple methods
  • I can copy and paste between applications
  • I understand the difference between copy and cut
  • I can undo mistakes confidently
  • I can gather research from web sources into notes efficiently
