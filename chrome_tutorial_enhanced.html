Split Screen & Window Management

MEDIUM
View multiple windows side-by-side for efficient multitasking

Why Split Screen?

Split screen lets you view two windows side-by-side. This is perfect for:

  • Comparing two documents or web pages
  • Taking notes while watching a video
  • Copying information from one window to another
  • Following instructions while working

üçé Mac Window Management Shortcuts

  • ‚åò + M - Minimize current window
  • ‚åò + H - Hide current app
  • ‚åò + Tab - Switch between apps
  • ‚åò + ` - Switch between windows of same app
  • Hover over green button ‚Üí Choose "Tile Window to Left/Right of Screen"

ü™ü Windows Snap Shortcuts

  • ‚äû Win + ‚Üê - Snap window to left half
  • ‚äû Win + ‚Üí - Snap window to right half
  • ‚äû Win + ‚Üë - Maximize window
  • ‚äû Win + ‚Üì - Minimize/Restore window
  • Alt + Tab - Switch between windows

üçé Mac Split Screen Using Mouse

  1. Hover over the green maximize button (top-left corner)
  2. Wait for a menu to appear
  3. Select:
    • "Tile Window to Left of Screen" or
    • "Tile Window to Right of Screen"
  4. Click the second window you want to see

To exit: Press Esc or move cursor to top and click the green button again

ü™ü Windows Snap Using Mouse

  1. Click and hold the title bar of a window
  2. Drag it to the left or right edge of your screen
  3. Release when you see a transparent outline
  4. The window will snap to half the screen
  5. Click another window thumbnail to fill the other half

Or: Right-click the taskbar ‚Üí Select "Show windows side by side"

  • Windows: Click first window ‚Üí Press ‚äû Win+‚Üê ‚Üí Click second window ‚Üí Press ‚äû Win+‚Üí
Snap windows side-by-side using mouse:
  • Mac: Hover over green button ‚Üí Select "Tile Window to Left" ‚Üí Click second window
  • Windows: Drag first window to left edge until it snaps ‚Üí Click second window to fill right side
