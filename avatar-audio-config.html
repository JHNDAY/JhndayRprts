Reference Notes
⚠ Why Two Separate Channels
Using BlackHole 2ch for one direction and BlackHole 16ch for the other prevents feedback loops. If both Zoom input and output used the same BlackHole device, audio would loop back on itself endlessly.
⚠ No Local Monitoring
With Zoom's speaker set to BlackHole 2ch, no audio plays through the Mac's speakers. This machine is a pure pass-through. Monitor the meeting from the host PC or another device.
ℹ Original Sound for Musicians
Set Zoom's Microphone Mode to Original Sound for Musicians to disable Zoom's noise suppression and audio processing. This prevents Zoom from filtering out or compressing Claude's TTS voice before it reaches participants.
ℹ Chrome Follows System Output
Chrome does not have a separate audio output selector — it uses the Mac system output device automatically. Setting system output to BlackHole 16ch is sufficient to route Claude's browser TTS into Zoom.
