üîß Troubleshooting

Common Issues & Solutions

‚ùå Page isn't loading or looks broken

Solution: Refresh the page (F5 or Ctrl+R). Make sure you're using a modern browser. Try opening in a different browser if the problem persists.

‚ùå Filters aren't working

Solution: Clear your filters (set all to "All"). Refresh the page. Make sure JavaScript is enabled in your browser settings.

‚ùå Search isn't finding courses I know exist

Solution: Check your spelling. Try broader terms (search "music" instead of "musicology"). Clear any active filters that might be hiding results.

‚ùå Course cards won't expand

Solution: Make sure you're clicking on the card itself, not just the "View Bio" link. Try refreshing the page. Ensure JavaScript is enabled.

‚ùå Biography links aren't working

Solution: Click the "View Bio ‚Üí" link next to instructor names. The page should scroll to that bio and highlight it briefly. If not, scroll down to the biography section manually.

‚ùå Statistics showing 0 courses

Solution: You likely have filter combinations that match no courses. Try removing some filters or selecting "All" for each filter category.

‚ùå Text is too small/large

Solution: Use your browser's zoom: Ctrl+Plus to zoom in, Ctrl+Minus to zoom out, Ctrl+0 to reset. On mobile, use pinch gestures to zoom.

‚ùå Can't find a specific instructor's bio

Solution: Use the search box in the Biography section. Type the instructor's name. Bios are alphabetical‚Äîscroll through the section to browse.

‚ùì Frequently Asked Questions

Q: How many courses are in the catalog?

A: The Spring 2026 catalog contains over 100 courses across 5 days, multiple periods, and 12+ subject categories.

Q: Can I print the catalog?

A: Yes! Use your browser's print function (Ctrl+P or Cmd+P). Consider printing only expanded sections you're interested in to save paper.

Q: Can I save my favorite courses?

A: The catalog doesn't have a built-in save feature, but you can: bookmark the page, take screenshots of courses you like, or write down course codes to reference later.

Q: What does "No Class April 6" mean?

A: Some courses note dates when class won't meet (usually holidays or breaks). The course still runs the stated number of weeks, just with a skip week.

Q: What's the difference between "5a" and "5b" courses?

A: 5-week courses are split into two sessions per semester: "5a" meets in the first 5 weeks, "5b" meets in the second 5 weeks. This lets you take two 5-week courses in the same time slot!

Q: Can I filter by multiple categories at once?

A: The dropdown filters select one option at a time, but you can use the search box to find courses across multiple topics (e.g., search "music history" to find courses in both categories).

Q: How do I know if I need previous experience for a course?

A: Read the full course description (click to expand). Most courses indicate if prerequisites or prior knowledge is needed. Many state "No prior experience required" explicitly.

Q: What does "Preparation Time" mean?

A: This is the estimated weekly time needed outside of class for homework, reading, watching videos, or other preparation. It helps you plan your schedule and workload.

Q: Can I take courses in different modalities?

A: Absolutely! You can mix in-person, Zoom, and hybrid courses. Just make sure the time periods don't conflict.

Q: How do I actually register for courses?

A: This catalog is for browsing and planning. For registration information, see the BOLLI Website referenced in the catalog header, or contact bolli@brandeis.edu.

Q: Will this catalog be updated?

A: This is a static HTML file showing the Spring 2026 offerings as published. For the most current information or any changes, check the official BOLLI website.

Q: Can I share this catalog with friends?

A: Yes! The HTML file is self-contained and can be shared. Just send the file or provide the link if it's hosted online.
